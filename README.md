# Sistema de Gestão de Cadastro

## 📌 Sobre o Projeto
Este sistema web foi criado para auxiliar o cadastro de pessoas e produtos, guardando os dados em uma tabela organizada e permitindo alterações a qualquer momento. Além disso, possui um sistema de **internacionalização (i18n)**, permitindo alterar todo o site para o idioma escolhido instantaneamente.

## 🚀 Funcionalidades
- **CRUD Completo:** Cadastrar e alterar pessoas e produtos no banco de dados.
- **Internacionalização:** Troca dinâmica de idiomas em todas as páginas.
- **Autenticação:** Acesso restrito ao sistema por meio de login único.
- **Gestão de Dados:** Visualização clara e editável através de tabelas dinâmicas.

## 🛠️ Tecnologias Utilizadas
- **HTML5**
- **CSS3**
- **PHP**
- **MySQL** (Banco de Dados)
