# Registration Management System

## 📌 About the Project
This web-based system was developed to streamline the registration of people and products. It stores data in a structured table, allowing for real-time updates and management. Additionally, it features a complete **internationalization (i18n)** system, enabling the user to switch the entire interface to their preferred language.

## 🚀 Features
- **Full CRUD:** Register, view, and edit people and products in the database.
- **Internationalization:** Instant language switching across the entire site.
- **Authentication:** Secure system access via a unique user login.
- **Dynamic Table:** Organized data display for easy management.

## 🛠️ Technologies Used 
- **HTML5**
- **CSS3**
- **PHP**
- **MySQL** (Database)
